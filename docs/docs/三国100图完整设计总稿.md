# 三国100图完整设计总稿

## 1. 项目定位

本方案基于你当前已上架的 `game_djbtz` C/S 小游戏框架，重构为：

- 三国题材推图养成小游戏
- 核心乐趣参考《风云天下OL》的推图、三围、武将独特技能、战法、普攻流、法攻流
- 轻量化回合制自动战斗
- 100 张大图推进
- 每图 24 个节点
- 击败守将后可解锁招募
- 可持续接入你现有的聊天、邮件、排行、广告、活动、商城与付费体系

一句话定位：

`100图长线推图 + 三国武将三围成长 + 特殊武将进阶 + 轻量C/S自动战斗小游戏`

## 3. 战斗与三围设计

### 3.1 阵容

- 上阵人数：`5`
- 站位：`前排2 + 后排3`
- 自动回合制
- 20 回合超时判负优先按剩余总兵力判定

### 3.2 三围定义

- `统`：影响普攻伤害、普攻减伤、反击强度
- `勇`：影响战法伤害、战法减伤、暴击收益
- `智`：影响法攻伤害、法术减伤、控制命中

### 3.3 流派

- 普攻流：依赖统，追击、反击、连击
- 战法流：依赖勇，高爆发、纵排、破甲、斩杀
- 法攻流：依赖智，灼烧、减怒、沉默、冰冻
- 辅助流：治疗、护盾、净化、回怒

### 3.4 士气

- 初始士气：0
- 普攻命中：+25
- 被攻击：+15
- 击杀：+30
- 满100自动放战法

## 4. 掉落与招募规则

## 4.0 推图体力与恢复规则

### 4.0.1 基础体力

- 体力上限：`120`
- 自然恢复：`每6分钟恢复1点`
- 12小时可自然恢复满体力
- 体力不允许溢出自然恢复

### 4.0.2 每日补给

- 午间补给：`12:00-14:00` 领取 `40体力`
- 晚间补给：`18:00-20:00` 领取 `40体力`

### 4.0.3 元宝购买体力

| 次数 | 消耗 | 获得 |
|---|---:|---:|
| 第1次 | 50元宝 | 60体力 |
| 第2次 | 80元宝 | 60体力 |
| 第3次 | 100元宝 | 60体力 |
| 第4次 | 120元宝 | 60体力 |

- 每日购买上限：`4次`

### 4.0.4 额外来源

- 好友互送：`每日30体力`
- 广告恢复：`每日2次，每次30体力`
- 月卡额外：`每日领取30体力`

### 4.0.5 节点消耗

- 普通节点：`5体力`
- 精英节点：`8体力`
- Boss节点：`10体力`
- 事件点：`0体力`
- 宝箱点：`0体力`

### 4.0.6 单图体力测算

以 `24节点` 的标准图计算：

- 普通节点 `16 × 5 = 80`
- 精英节点 `4 × 8 = 32`
- Boss节点 `1 × 10 = 10`
- 事件点 `2 × 0 = 0`
- 宝箱点 `1 × 0 = 0`

单图合计约：`122体力`

结论：

- 轻度玩家约可在一天内完成一张完整大图
- 活跃玩家可通过补给、好友、广告多推进半张到一张图
- 付费玩家只能加速，不会无限推图

### 4.1 节点掉落

- 普通节点：铜钱、主公经验、经验酒、装备碎片、进阶石
- 精英节点：额外掉武将碎片、突破丹、宝物精华
- Boss节点：固定掉 Boss 碎片、招募令、高阶装备材料

### 4.2 击败守将后可招募

- 普通 Boss：解锁酒馆招募权重
- 精英 Boss：解锁精英扫荡碎片
- 关键 Boss：首通送 `5-10碎片`
- 阶段核心 Boss：首通送 `30碎片` 或整将

### 4.3 掉落池分层

- 前期：铜钱、经验酒、蓝装碎片、招募令碎片、进阶石
- 中期：紫装碎片、突破丹、武将碎片、宝物精华
- 后期：橙装碎片、红装碎片、将魂、神兵石
- 终局：金装碎片、神魔结晶、觉醒石、天命令、专属宝物材料

## 15. 对比《风云天下OL》仍缺少的核心设计

当前这份方案已经有了：

- 100图长线推图
- 武将三围
- 普攻流、战法流、法攻流
- Boss掉落与击败后招募
- 特殊武将进阶

但如果严格对比《风云天下OL》的“耐玩感”，还缺下面这些关键层：

### 15.1 装备系统深度不够

目前方案只有装备槽位概念，但还缺：

- 装备分部位的属性分工
- 套装效果
- 强化、精炼、洗练
- 稀有装备追求

### 15.2 阵法系统缺失

《风云天下OL》这类游戏里，阵法是推图卡点的重要突破口。  
我们现在有“站位”，但没有“阵法”，深度不够。

### 15.3 兵种克制系统不完整

目前小兵和武将只有名字和流派，没有明确兵种关系。  
少了兵种后，推图会更像纯数值对撞。

### 15.4 武将羁绊和国家队体系偏弱

目前只有少量羁绊举例，还没有形成：

- 国家阵营加成
- 核心羁绊树
- 同名进阶路线的战术变化

### 15.5 资源副本和功能副本不足

只有主线图不够，需要独立副本承接养成材料：

- 装备本
- 将魂本
- 宝物本
- 突破本
- 金币本
- 经验本

### 15.6 PVE长期目标不足

主线推图之外，还缺：

- 爬塔
- 名将试炼
- 世界Boss
- 限时活动Boss

这些会直接影响留存。

### 15.7 PVP纵深不足

目前只有竞技概念，缺少：

- 防守阵容
- 段位奖励
- 赛季结算
- 巅峰禁将

### 15.8 数值循环缺少“第二成长轴”

现在主要成长轴是武将本体。  
还需要第二成长轴来托住中后期：

- 装备
- 阵法
- 宝物
- 军团科技
- 图鉴

## 16. 补强后的完整系统框架

下面是建议补上的完整系统。

### 16.1 装备系统

#### 16.1.1 部位

- 武器
- 头盔
- 战甲
- 坐骑
- 兵书
- 兵符

#### 16.1.2 属性分工

- 武器：普攻攻击、暴击
- 头盔：兵力、物防
- 战甲：兵力、战法减伤
- 坐骑：速度、战法攻击
- 兵书：法强、控制命中
- 兵符：兵力上限、统勇智百分比

#### 16.1.3 品质

- 白
- 绿
- 蓝
- 紫
- 橙
- 红
- 金

#### 16.1.4 养成线

- 强化：固定加基础属性
- 精炼：加百分比属性
- 洗练：随机副词条
- 套装：2件/4件/6件效果

#### 16.1.5 推荐套装

| 套装 | 2件效果 | 4件效果 | 6件效果 |
|---|---|---|---|
| 虎啸 | 普攻+8% | 暴击+10% | 普攻后20%追击 |
| 破军 | 战法+10% | 首次战法增伤15% | 破甲2回合 |
| 天机 | 法伤+10% | 控制命中+12% | 法攻后减怒10 |
| 玄武 | 兵力+12% | 减伤+8% | 开场护盾15% |
| 回春 | 治疗+12% | 受疗+10% | 回合末最低血回血 |

### 16.2 阵法系统

#### 16.2.1 设计目标

阵法要成为“卡图后的新解法”，而不是单纯战力加成。

#### 16.2.2 阵法列表

| 阵法 | 定位 | 效果 |
|---|---|---|
| 锋矢阵 | 爆发 | 后排主C增伤20%，前排减伤-8% |
| 鱼鳞阵 | 坦度 | 前排减伤18%，后排输出-6% |
| 雁行阵 | 法攻 | 后排法伤+18%，速度+8 |
| 长蛇阵 | 单点突破 | 纵排增伤15%，单体战法+12% |
| 方圆阵 | 均衡 | 全队兵力+10%，全抗性+6% |
| 八卦阵 | 控制 | 智+10%，控制命中+12% |
| 虎翼阵 | 普攻 | 普攻暴击+12%，追击率+8% |
| 云垂阵 | 收割 | 对半血以下目标增伤18% |

#### 16.2.3 阵法成长

- 1级解锁基础效果
- 5级解锁副属性
- 10级解锁特殊效果
- 阵法材料来源：
  - 阵法塔
  - 活动商店
  - 军团商店

### 16.3 兵种系统

#### 16.3.1 武将兵种

每个武将固定一个兵种：

- 枪兵
- 骑兵
- 盾兵
- 弓兵
- 谋士
- 方士

#### 16.3.2 克制关系

```text
骑兵克弓兵
弓兵克枪兵
枪兵克骑兵
盾兵克普攻型前排
谋士克高统低智目标
方士克重甲目标
```

#### 16.3.3 克制数值

- 克制增伤：`15%`
- 被克减伤：`-10%`
- 终局系统可扩到 `20%`

#### 16.4.1 羁绊类型

- 国家羁绊
- 人物关系羁绊
- 宿敌羁绊
- 师徒/君臣羁绊

### 16.10 精英副本系统

主线图之外，增加：

- 每张图的 `4个精英节点` 独立进入
- 主要产出本图核心守将碎片

这样“击败守将可招募”才能变成长期可养成，而不是一次性解锁。

### 16.11 竞技场系统补强

#### 16.11.1 基础竞技场

- 挑战 5 次/天
- 防守阵容独立配置
- 段位：
  - 青铜
  - 白银
  - 黄金
  - 铂金
  - 钻石
  - 王者

#### 16.11.2 巅峰竞技

- 周末开启
- 红将以上可上
- 每周禁用 3 名热门武将
- 奖励神魔材料

- 战法伤害%
- 法伤%
- 控制抗性%
- 世界Boss增伤%

### 16.13 活动体系补强

#### 16.13.1 限时神将活动

- 7天或14天一期
- 活动核心：
  - 活动副本
  - 活动Boss
  - 活动商店
  - 活动任务
  - 活动卡池

#### 16.13.2 特殊武将进阶活动

例如：

- 吕布 -> 魔吕布 -> 神吕布
- 关羽 -> 武圣关羽 -> 真武关羽
- 诸葛亮 -> 神机诸葛

这些必须通过活动持续拉长生命周期。

目前文档里有数值和UI，但表现层还缺：

- 战法演出分级
- 暴击镜头

- 控制状态图标
- 回怒条
- 伤害类型区分动效

#### 16.14.1 建议分级

- 普通将：简短战法特效
- 橙将：带武将名条
- 红将：全屏半幅战法演出
- 金将：全屏动态背景切换

## 33. 各系统数值联动校验

本节用于检查不同系统是否会出现数值冲突、成长断档或付费溢出。

## 33.1 战斗成长轴校验

当前主成长轴共 6 条：

1. 武将等级
2. 武将星级/觉醒
3. 装备强化/精炼/洗练
4. 阵法
5. 宝物
6. 羁绊/图鉴/军团科技

校验结论：

- 前期 `1-20图` 主要依赖：
  - 武将等级
  - 基础装备
  - 基础技能等级
- 中期 `21-60图` 主要依赖：
  - 武将升星
  - 紫橙装
  - 阵法1-5级
  - 宝物开放
- 后期 `61-90图` 主要依赖：
  - 红装
  - 阵法6-10级
  - 觉醒
  - 羁绊完整
- 终局 `91-100图` 主要依赖：
  - 金装碎片
  - 神魔结晶
  - 专属宝物
  - 神魔武将进阶

结论：

- 成长轴层次清晰，没有“中期只能硬升主角等级”的断层
- 终局内容不会只靠单一养成项碾压

## 33.2 推荐战力与产出匹配校验

校验目标：

- 玩家在正常日活下，`3-5天` 需要发生一次明显提升
- 主线不应要求玩家长期停留同一图超过 `2-3天`

校验结果：

- `1-10图`：一天可推进完成
- `11-30图`：2-4天内推进
- `31-60图`：约1-2周
- `61-90图`：约2-4周
- `91-100图`：作为赛季后期目标合理

结论：

- 长度适合小游戏产品，不会前期太短、后期完全无事可做

## 33.3 装备与武将成长占比校验

建议中期战力构成：

| 来源 | 占比 |
|---|---:|
| 武将等级/星级 | 45% |
| 装备 | 22% |
| 阵法 | 10% |
| 宝物 | 8% |
| 羁绊/图鉴 | 7% |
| 军团科技/其他 | 8% |

结论：

- 武将本体仍是核心
- 装备和阵法足够重要，能成为卡图解法
- 不会出现“只看装备不看武将”的偏移

## 33.4 货币产消平衡校验

校验目标：

- 铜钱必须短缺但不至于卡死
- 元宝必须稀缺但每日有稳定获取
- 将魂和神魔结晶必须稀有，形成长期追求

结论：

- 铜钱：日常产出约50万，中期单日主力消耗约25万到40万，合理
- 元宝：日常产出约280，足够少量抽卡和买体力，合理
- 将魂：单日约360，只够主养1到2个武将，合理
- 神魔结晶：仅靠活动和Boss，符合终局定位

## 33.5 付费深度校验

当前设计中：

- 白嫖可完成主线和中段竞技
- 月卡玩家拥有明显但不破坏平衡的加速
- 神魔将需要活动积累，不是单纯直购即毕业

结论：

- 付费拉开效率，不直接跳过所有策略
- 仍符合《风云天下OL》式“阵容理解”乐趣
