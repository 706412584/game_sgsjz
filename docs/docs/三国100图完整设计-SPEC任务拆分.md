本文件把总稿拆成可执行任务，并使用以下状态表示“实际开发状态”：

- `[x]` 已完成：已经实际完成开发或交付
- `[-]` 进行中：已经开始开发，尚未完成
- `[ ]` 未开始：尚未进入实际开发
